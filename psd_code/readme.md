# psd_code文件夹
 
这是计算psd的主要python文件。
- [ ] eeg_psd_anova.py
  - [ ] 主要用来计算psd的anova
- [ ] eeg_psd_channel.py
  - [ ] 主要用来挑选差异比较大的通道
- [ ] eeg_psd_csv.py
    - [ ] 主要用来计算alpha1、alpha2、beta、gamma、delta等psd并存储csv文件
 - [ ] eeg_psd_plot.py
    - [ ] 用来画psd的图像
 - [ ] eeg_svm_classify.py
    - [ ] svm的分类器，基本弃用。
    - [ ] 分类可以使用classify目录下的eeg_classify_model.py
 - [ ] eegAnalyze.py
    - [ ] 用来读取eeg信息的文件
